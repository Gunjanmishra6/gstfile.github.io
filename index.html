About GST

Goods and Services Tax (GST) is an indirect tax applied to the supply of goods and services. It replaces multiple cascading taxes levied by the government.

Types of GST Returns

  • GSTR-1 – Details of outward supplies
  • GSTR-3B – Monthly summary return
  • GSTR-9 – Annual return

GST Filing Process

  1. Login to the GST Portal
  2. Select the return type
  3. Enter invoice details
  4. Verify tax amounts
  5. Submit and file with DSC/EVC
